## Publications

**Agnostic Sample Compression Schemes for Regression**
Establishing agnostic sample compression schemes for regression with near-optimal rates. *Published at ICML 24 (Spotlight)*

**Relaxed Models for Adversarial Streaming: The Advice Model and the Bounded Interruptions Model**
Introducing relaxed adversarial streaming models that allow practical algorithm design with provable guarantees. *Published at ESA 23*

**Adaptive Data Analysis with Correlated Observations**
A fundamental study regarding the possible extensions of adaptive data analysis tools such that they could be used with correlated observations. *Published at ICML 22*

**On the Sample Complexity of Privately Learning Axis-Aligned Rectangles**
A privacy preserving learning algorithm for the Axis-Aligned Rectangles problem which has near-optimal sample complexity. *Published at NeurIPS 21*

**Sample Compression for Real-Valued Learners**
An algorithmically efficient version of the learner-to-compression scheme conversion which is also extended to regression hypotheses classes. *Published at ALT 19*
